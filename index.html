The Basics of Web and HTML

The web is comprised of users, web browsers, the cloud and servers. The componnents communicate using HTTP requests to access Hypertext Markup Language.

Hypertext Markup Language (HTML)

HTML documents make up the primary content of the web. Tags and elements are used to define the content and how it should be displayed.

Information can be displayed inline or block.

Inline elements include:

A
SPAN
BR
IMG
STRONG

Block elements include:

DIV
P
FORM
